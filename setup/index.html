Software updates

This software is sold as-is, with no guarantees for new feature updates. The developer does guarantee bug-fix updates to address any failing or broken features while you are within an active support period.

Updates are delivered to you over the air. The script comes with an optional automatic update system that can install updates for you. Alternatively, you may download an archive of each update's modified files through the admin panel.

Data collection

The developer of the script may, depending on your configuration, collect data from your application. This data is limited to:

  • Details about errors that occur in the script.
  • Statistics on how often certain features and tools are used.
  • Basic server information, such as your PHP version and extensions.

You will be given a chance to review these settings in the installer. After installing, you can also change these settings from the admin panel.

We respect your privacy. The data we collect is used only to improve our software. For instance, we use telemetry to determine which features are used most often, and focus our development on improving those features first.
